# Servicios que ofrece syrtix.com

## Servicios Principales (Core Services)

### Diseño Web & UX/UI
- **Diseño web profesional** - Interfaz y experiencia de usuario
- **Logo design** - Creación de identidad visual
- **Identidad visual completa** - Logo + paleta de colores + tipografía + manual de marca
- **Prototipado y wireframing** - Diseño de flujos y estructura antes de desarrollar
- **Diseño accesible (a11y)** - Cumplimiento WCAG, contraste, navegación inclusiva

### Rebranding & Estrategia
- **Rebranding completo** - Auditoría + identidad visual + rediseño web
- **Refreshing de marca** - Actualización ligera manteniendo esencia
- **Análisis y consultoría de marca** - Definir posicionamiento y dirección estratégica

### Estrategia & Análisis
- **Auditoría UX + Testing de usabilidad** - Análisis de problemas con usuarios reales y recomendaciones
- **Optimización de conversiones (CRO)** - A/B testing y mejora de flujos para aumentar ventas
- **Análisis competitivo** - Benchmarking de competencia para posicionamiento estratégico

### Desarrollo Web
- **Landing pages** - Páginas de conversión optimizadas
- **Páginas web** - Sitios corporativos, blogs, noticias, educacionles
- **Aplicaciones web** - Herramientas B2B complejas, dashboards, CRM, ERP
- **Portales con membresía y comunidades** - Sitios con login, roles, contenido exclusivo
- **Sitios multiidioma (i18n)** - Soporte para múltiples idiomas y regiones
- **Sistemas de reservas** - Para hoteles, spas, clínicas, servicios profesionales

### E-commerce & Marketplaces
- **Tiendas online básicas** - Hasta X productos con carrito, pagos y gestión de pedidos
- **Tiendas online avanzadas** - Inventario dinámico, múltiples proveedores, automatizaciones
- **Integraciones de pagos** - Stripe, PayPal, criptomonedas, plan de suscripciones
- **Gestión de inventario** - Sincronización automática de stock, alertas de bajo inventario
- **Integración con plataformas de envío** - Seguimiento automático de pedidos, etiquetas
- **Programas de afiliados y referidos** - Sistema de comisiones automáticas
- **Análisis y reportes de e-commerce** - Dashboards de ventas, productos top, ROI
- **[PREMIUM] Plataformas multi-vendedor** - Tipo Etsy, MercadoLibre (vendedores + compradores)
- **[PREMIUM] Comisiones y pagos automáticos** - Gestión de ingresos por vendedor

### Aplicaciones Móviles
- **Desarrollo iOS & Android nativo** - Apps nativas con máximo rendimiento
- **Apps híbridas (React Native/Flutter)** - Una base de código para iOS y Android
- **Progressive Web Apps (PWA)** - Apps instalables desde el navegador
- **MVP y prototipos rápidos** - Desarrollo ágil para validar ideas
- **Integración de geolocalización** - Maps, GPS, servicios basados en ubicación
- **Push notifications** - Notificaciones en tiempo real y segmentadas
- **In-app purchases y monetización** - Sistema de pagos y suscripciones dentro de la app
- **Integración de cámara y sensores** - Foto, video, biometría, acelerómetro
- **Sincronización offline** - Funcionalidad sin conexión con sincronización
- **App Store Optimization (ASO)** - Optimización para posicionamiento en tiendas
- **Testing y Quality Assurance** - Pruebas exhaustivas en múltiples dispositivos
- **Integración de Firebase o backend personalizado** - Base de datos, autenticación, analytics
- **Mantenimiento y updates** - Soporte continuo, actualizaciones, bug fixes

- **Auditoría UX/UI** - Testing con usuarios, mapas de calor, comportamiento
- **Optimización de rendimiento** - Velocidad de carga, Core Web Vitals, Lighthouse
